React中context的使用

context

context提供了一种方式，能够让数据在组件中传递时不必一级一级的手动传递

一般情况下，数据在组件中，要一级一级的传递，单向数据流，比如Parent组件中的theme值，需要在Item组件中使用，就需要我们从Parent中向下传递， 但当我们有了Context后，我们就不需要一级一级传递了
